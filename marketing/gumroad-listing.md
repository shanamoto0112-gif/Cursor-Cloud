## 3行サマリ

Excel で名簿や注文リストを扱う前に、CSV の重複行をコマンド1つで除去。  
データは PC 内だけで処理され、外部送信なし。日本語 CSV（Shift_JIS）にも対応。  
ダウンロード後すぐ使える exe + 日本語 README 付き。

---

## 商品説明（Gumroad 本文）

### こんな人向け

- 名簿・顧客リスト・イベント参加者 CSV の重複を手作業で探している
- 無料の Web ツールにデータをアップロードしたくない
- Excel を開く前にサッと重複除去したい

### できること

- CSV の重複行を一括削除
- 「メールアドレス列だけ」などキー列を指定して重複判定
- 最初の行 / 最後の行、どちらを残すか選択可能
- UTF-8・Shift_JIS を自動判定

### 含まれるもの

- `dedupcsv.exe`（Windows 10/11）
- `README.md`（日本語の使い方）
- サンプル CSV

### 使い方（例）

```
dedupcsv.exe run input.csv output.csv --keys email
```
